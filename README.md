# code-feedback

## 插件介绍
- 选择代码片段
- 将代码片段高亮展示
- 获取代码片段的参与者
- 获取提交反馈的Git用户名
- 提交代码片段

## 打开插件
- 通过ctrl + shift + p 唤起指令输出`openCodeSnippetFeedbackPanel`，即可快速打开插件
- 通过左侧菜单栏找到`代码片段反馈`的功能图标，点击即可打开插件
